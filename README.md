# Spiritbit - Lightweight eBPF Zero Trust EDR

**Ultra-lightweight kernel-level security watchdog for Linux

- **RAM Usage**: < 20MB
- **7-Day Learning Period** with automatic protection mode
- **Advanced Behavioral Analysis** + ML Telemetry
- **Unstoppable Self-Protection** (dual watchdog, stealth mode)

![License](https://img.shields.io/badge/License-GPLv3-blue.svg)
![Language](https://img.shields.io/badge/Language-C-blue)

## Features

- Full filesystem + process + network monitoring via eBPF
- 8D behavioral scoring engine
- 7-Day automatic learning period
- Process freezing & quarantine
- Advanced ML telemetry
- Rate limiting & flood protection
- Self-healing dual watchdog system
